j'ai passer la majeures partie de mon stages à contribuer dans le project des ruches connecter.
Ma mission principalle étais de consevoir le code permetant de faire fontionner les ruches.
Tout d'abore l'on m'as expliquer le concepte de ces ruchen, j'ai compris quelle avais pour but d'aider les abiculteurs. En recoltant des donner via des capteurs(ses données peuvent aller de la température , poid, quantiter de miel dans les ruchen, humiditer ) Ces donner aide non seulement les abiculteurs mais aussi nous aide à approfondir notre compréantion sur les comportament des abailles.
les ruches sont entierement consue dans l'établissement, de la conception des pieces, leurs création via une imprimentes 3D anci qu'à la constructions.
Pour la partie electronique il y à plusieurs modules utiliser : capteur , les transmition, recupération , envoie sur le web.
on avais un module pour les capteurs ainci qu'une carte arduino afin de traiter et collecter les donner. ce carte permet de recupere les info du capteur mais aussi de le convertir en un octet afin de pouvoir faciliter la transmition.
