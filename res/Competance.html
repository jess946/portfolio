    languages connues

    php

    Hypertext Preprocessor,est un langage de programmation libre, pouvant également fonctionner comme n'importe quel langage interprété de façon locale, principalement utilisé pour produire des pages Web dynamiques.

    java

    Java est un langage de programmation orienté objet. La plateforme Java permet peut se soustraire du système d'exploitation, ce dernier faisant la liaison. Ainsi une même application Java peut tourner sur Windows ou Linux.

    html

    Le HyperText Markup Language, généralement abrégé HTML ou, dans sa dernière version, HTML5, est le langage de balisage conçu pour représenter les pages web.

    python

    Python a été conçu pour être un langage lisible. Par exemple, il possède moins de constructions syntaxiques que de nombreux langages structurés tels que Cou le java.

    symfony

    Est un framwork back-end crée à partir d'un ensemble de composants PHP. Il fournit des fonctionnalités modulables et adaptables qui permettent de faciliter et d’accélérer le développement d'un site web.

    angular

    Est un framework côté client, open source, basé sur TypeScript. L'avantage d'AngularJS est qu'il permet aux développeurs de traduire des documents HTML statiques en contenu dynamique.
